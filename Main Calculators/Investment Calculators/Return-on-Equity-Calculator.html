Return on Equity Calculator

What is Return on equity?

Return on equity (ROE) may be a measure of monetary performance calculated by dividing net by shareholders' equity. Because shareholders' equity is adequate to a company’s assets minus its debt, ROE is taken into account the return on net assets.

Return on equity formula

ROE is expressed as a percentage and can be calculated for any company if net income and equity are both positive numbers.

ROE = Net Profit/Equity

How do you calculate return on equity?

Suppose a company with an annual income of $1,800,000 and average shareholders' equity of $12,000,000.

Then Return on Equity value is calculated as:

ROE = $1800,000/$12000000 = 15%

Usage of Return on equity

• Sometimes an extremely high ROE is a good thing if net income is extremely large compared to equity because a company’s performance is so strong.

• A second issue that could cause a high ROE is excess debt. If a company has been borrowing aggressively, it can increase ROE because equity is equal to assets minus debt.

What is a good ROE?

As with return on capital, a ROE is a measure of management's ability to generate income from the equity available to it. ROEs of 15–20% are generally considered good. ROE is also a factor in stock valuation, in association with other financial ratios.

What is a bad return on equity?

Return on equity (ROE) is measured as net income divided by shareholders' equity. When a company incurs a loss, hence no net income, return on equity is negative. ... If net income is negative, free cash flow can be used instead to gain a better understanding of the company's financial situation.
